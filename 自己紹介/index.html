私はPythonやウェブ開発に興味があり、健康管理をサポートするアプリを開発中です。特に、医療分野で役立つソフトウェアに 強い関心があります。

将来の目標: 医療や介護に役立つアプリ開発者として成長し、より多くの人々の生活を改善したいと考えています。

スキルセット

  • プログラミング言語: Python, HTML, CSS, JavaScript
  • フレームワーク: Flask
  • データベース: SQLite
  • 開発ツール: VS Code, Git, GitHub
  • その他: Docker, Visual Studio Codeでのプロジェクト管理

成果物: 尿管理アプリ

このアプリは、医療現場や在宅介護で使用される患者の尿管理を効率化するために開発しました。ユーザーは毎日の尿量を記録し、カレンダーや統計機能を使って管理できます。

使用技術

  • フロントエンド: HTML, CSS, JavaScript
  • バックエンド: Python（Flask）
  • データベース: SQLite
  • セキュリティ: ログイン機能（ユーザー認証システム）

主な機能

  • 尿量記録の追加・削除
  • 日ごとの尿量の合計計算
  • カレンダー機能で記録を視覚化
  • ログイン・パスワード保護機能
  • データの保存と共有機能
